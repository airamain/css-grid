Comando de Git

- git clone, para clonar un repositorio
    Ej: git clone nombreRepo

- git status, nos muestra el estado actual de nuestra branch

- (1) git add, para agregar los cambios en nuestra rama o branch

- (2) git commit -m "COMENTARIO" (si van las comillas)

- git push origin rama/branch, para subir nuestro codigo a la rama/branch

- git pull origin rama/branch, para descargar a la rama/branch o los cambios

- git switch, es para ver los branch o ramas que tenemos.

- git checkout -B NombreDelBranch

- git checkout, sirve para revertir los cambios en un branch

- git merge, para unir una rama en otra
